# ModIssueTracker

Welcome to the official issue tracker for DoubtfulFanboy projects.

This repository is used to report bugs, crashes, compatibility issues for all supported projects.

## Before Creating an Issue

Please ensure that:

* You are using the latest version of the affected project.
* You have checked existing issues to avoid creating duplicates.
* You include any relevant logs, crash reports, screenshots, or reproduction steps.

## Supported Projects

* Bundle Expansion
* Isometric Chunk
* Nylium Nasturtium
* Glow Berry Lantern

Additional projects may be added in the future.

## Bug Reports

When reporting a bug, please provide:

* The affected project
* Minecraft version
* NeoForge version
* Mod version
* Steps to reproduce the issue
* Relevant logs or crash reports
